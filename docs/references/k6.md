## How Dilnova Uses k6

- Load test script: `scripts/load-test-k6.js`
- Targets **public read paths** only (homepage, product listing, contact, cart)
- Write paths are rate-limited and auth-gated, so excluded from load tests

### Run Commands

```bash
# Local
k6 run scripts/load-test-k6.js

# Against staging/production
TARGET_URL="https://www.dilstar.pp.ua" k6 run scripts/load-test-k6.js
```

### Performance Baselines

| Endpoint               | Target VUs | Target RPS | Target p95 Latency |
| ---------------------- | ---------- | ---------- | ------------------ |
| Home (`/`)             | 50         | 250        | < 120ms            |
| Products (`/products`) | 50         | 200        | < 150ms            |
| Contact (`/contact`)   | 50         | 50         | < 100ms            |
| Cart (`/cart`)         | 50         | 50         | < 100ms            |
